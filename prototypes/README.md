# Prototypes

This folder is for static UI prototypes and mockup reference files.

## Current Prototype

The current working prototype is:

```text
gt-orders-stocks-ui-prototype-v1-2-full-seed-data.html
```

This file was generated during the UI design phase and includes:

- Inventory page
- Customer page
- Sales Order page
- Local embedded seed data
- Search filters
- Mockup drawers
- Sales order list and detail layout

## Usage

Use the prototype as a visual reference when converting the project into a Next.js front-end.

The prototype is not the final app architecture.

## Important

Do not treat the embedded prototype data as a production database.

Use `/data` seed files for front-end implementation.
